# Flask Boilerplate

This repository is a boiler template code to create a Flask APP.


Checkout *requirements.txt* for libraries used.

**Versions**
Python: 3.9.6

**Installation**
* Install Python 3.9.6 or greater
* Install pip
* Create virtual env via pip -m venv
* Install dependencies
* Run Flask app via
  * flask run
  * flask run --reload
